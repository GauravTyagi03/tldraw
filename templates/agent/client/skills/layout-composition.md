- Group related items by proximity and leave clear gaps between unrelated groups.
- Use consistent alignment on shared edges or center lines when shapes are part of the same set.
- Prefer left-to-right or top-to-bottom flow for sequential information.
- Avoid packing shapes tightly. Leave whitespace so labels and connectors can breathe.
- If reorganizing existing content, move groups intentionally rather than making many tiny unrelated shifts.
