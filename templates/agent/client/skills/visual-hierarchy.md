- Make the primary idea visually dominant through size, position, grouping, or contrast.
- Keep supporting details smaller and spatially subordinate to the main idea.
- Use emphasis sparingly; if everything is highlighted, nothing is highlighted.
- Prefer one clear focal point per region of the canvas.
- Make the reading order obvious without requiring the user to inspect every shape.
