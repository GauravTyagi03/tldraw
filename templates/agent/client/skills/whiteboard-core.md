- Treat the canvas as a working surface, not a poster. Optimize for clarity, iteration, and reuse.
- Preserve useful existing structure unless the user asks for a redesign.
- Prefer simple shapes, direct labels, and clear spatial relationships over decoration.
- When the task is ambiguous, make the smallest useful improvement and explain what you changed.
- Use the existing tldraw actions and review loop to inspect and refine the result.
