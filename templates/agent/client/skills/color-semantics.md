- Use color to encode categories, status, or emphasis, not as random decoration.
- Keep the same concept the same color across the canvas.
- Use strong colors sparingly for warnings, contrast, or key takeaways.
- Do not rely on color alone; pair it with labels, position, or shape choice.
- Prefer a restrained palette when the user asks for clarity or organization.
